# Responsible Use

CivicSec Lab is designed for defensive, educational, and public-interest security work.

The project supports small organisations, NGOs, researchers, student teams, and civic technology groups in understanding risk signals and making better decisions. It is not an offensive security toolkit.

## Allowed Uses

- Learning about defensive security, privacy engineering, and civic-risk workflows.
- Analysing fictional or authorised organisational data.
- Reviewing public, documented vulnerability metadata for defensive triage.
- Building explainable risk dashboards and incident workflows.
- Testing privacy-risk scanning on synthetic or approved datasets.
- Monitoring aggregate narrative patterns from user-supplied public datasets.

## Prohibited Uses

- Exploiting systems or providing exploit instructions.
- Credential harvesting or phishing automation.
- Malware, persistence, evasion, or privilege escalation tooling.
- Targeting private individuals or creating harassment lists.
- Uploading real sensitive data to public demo instances.
- Using the platform to make irreversible decisions without human review.

## Human Review

CivicSec Lab should surface signals, evidence, confidence levels, and recommended actions. It should not claim certainty where evidence is incomplete.

Use outputs as decision support, not as automatic proof that a person, group, account, system, or organisation is malicious or unsafe.

## Sample Data

All sample data in this repository is fictional and created for demonstration purposes.
