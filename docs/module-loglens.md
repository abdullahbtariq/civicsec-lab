# LogLens

LogLens will analyse authorised or synthetic login logs for suspicious patterns.

## Future Scope

- CSV ingestion for demo login logs.
- Failed login bursts.
- New device or unusual location signals.
- Suspicious success after repeated failures.
- Evidence-backed RiskEvent creation.

## Safety Boundary

LogLens must not collect credentials, bypass authentication, or monitor systems without authorisation.
