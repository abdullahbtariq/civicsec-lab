# IncidentFlow

IncidentFlow will support lightweight incident response workflows for civic organisations.

## Future Scope

- Incident records.
- Playbook templates.
- Response tasks.
- Timeline notes.
- Report generation.
- Linkage to RiskEvents and evidence.

## Safety Boundary

IncidentFlow should support calm documentation, review, and recovery workflows. It should not encourage retaliation or offensive action.

## Phase 0 Status

Placeholder documentation only. No incident models or workflow logic exists yet.
