# DataPrivacy Doctor

DataPrivacy Doctor will help identify privacy risks in datasets.

## Future Scope

- Secure CSV upload.
- Column profiling.
- Direct identifier and quasi-identifier detection.
- Masked examples.
- Retention controls.
- Privacy-risk findings and reports.

## Safety Boundary

Public demos must not be used with real sensitive data. The tool should minimise raw data exposure and avoid claiming that anonymisation is guaranteed.

## Phase 0 Status

Placeholder documentation only. No upload or scanning logic exists yet.
